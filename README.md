# Exercice-01-Packet-Tracer

Après avoir placé tous les équipements demandés, j'ai placé les modules CFE/CGE sur les switch PT.
Ensuite j'ai relié chaque équipement entre eux : <br>
Routeur FAI -> Routeur R1<br>
Routeur R1 -> Switch S1<br>
Switch S1 -> Switch S2<br>
Switch S2 -> Switch S3<br>

Chaque switch représente un bureau et contient 2 PC fixes, 1 PC portable, 1 point d'accès et un téléphone. Tout les équipements du bureau sont relié au switch sauf le PC portable qui est connecté en wifi sur le point d'accès.

Ensuite j'ai configuré le routeur (voir config routeur R1), les switch (voir config switch), j'ai ajouté le modules WPC300N sur les PC portable puis j'ai paramétré le point d'accès pour que les PC portable se connectent sur le point d'accès de leur bureau et pas un autre.
